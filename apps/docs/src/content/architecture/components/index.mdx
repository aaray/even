# Component Architecture

The EVEN platform is organized as a monorepo with multiple applications and shared packages. This page provides an overview of all components and their relationships.

## Applications

### Frontend Applications

| App | Package | Port | Description |
|-----|---------|------|-------------|
| [Web App](/architecture/components/web-app) | `@even/web` | 3000 | Main artist dashboard |
| Storybook | `@even/storybook` | 6006 | Component documentation |
| [Docs](/architecture/components/web-app) | `@even/docs` | 3001 | This documentation site |

### Backend Applications

| App | Package | Port | Description |
|-----|---------|------|-------------|
| [API Server](/architecture/components/api) | `@even/api` | 4000 | REST API endpoints |

## Shared Packages

| Package | Description |
|---------|-------------|
| [`@even/ui`](/architecture/components/shared-packages) | Reusable UI components (shadcn/ui based) |
| [`@even/shared`](/architecture/components/shared-packages) | TypeScript types and Zod schemas |
| `@even/docs-content` | Documentation assets (diagrams, images) |

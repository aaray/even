# Web App (`@even/web`)

The Web App is the main frontend application, providing the artist dashboard interface.

## Technology Stack

| Layer | Technology | Version |
|-------|------------|---------|
| Framework | Next.js | 15.x |
| UI Library | React | 19.x |
| Styling | Tailwind CSS | 3.4.x |
| Components | shadcn/ui | via @even/ui |
| Data Fetching | TanStack Query | 5.x |
| Charts | Recharts | 2.15.x |
| i18n | next-intl | 4.x |
| Animations | Framer Motion | 12.x |

## Directory Structure

```
apps/web/
```

## Key Features

### Dashboard
- KPI cards showing key metrics
- Earnings chart with time range selection
- Product grid with filtering and sorting
- Fan engagement metrics

### Multi-Language Support
- English and Spanish locales
- Language persistence via localStorage
- next-intl for routing and translations

### Responsive Design
- Mobile-first approach
- Breakpoints: 375px - 1440px
- WCAG 2.1 AA accessibility

## Data Flow

The web app fetches data from the API server using TanStack Query:

```tsx
// Example: Fetching artist data
const { data: artist } = useQuery({
  queryKey: ['artist'],
  queryFn: () => fetch('/api/artist').then(r => r.json()),
});
```

## Dependencies

The web app depends on:
- `@even/ui` - UI components
- `@even/shared` - Type definitions and schemas
- `@even/api` - Backend API (at runtime)

## Configuration

Key environment variables:

| Variable | Description | Default |
|----------|-------------|---------|
| `NEXT_PUBLIC_API_URL` | API server URL | `http://localhost:4000` |

# Getting Started with EVEN

Welcome to EVEN! This guide will walk you through the basics of using the Artist Dashboard.

**Estimated time**: 5 minutes
**Difficulty**: Beginner

## Prerequisites

- An active EVEN artist account
- Access to the Artist Dashboard

## Dashboard Overview

When you first log in, you'll see the main dashboard with:

1. **KPI Cards** - Key metrics at a glance (total earnings, fans, products)
2. **Earnings Chart** - Visual breakdown of your revenue over time
3. **Products Grid** - Your products with sales data
4. **Fan Engagement** - Metrics about your fan base
